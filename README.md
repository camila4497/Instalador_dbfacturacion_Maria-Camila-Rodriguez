# Sistema de Facturación y Administración (Cliente-Servidor)

### Instalación para el Usuario Final

El paquete de instalación fue creado con Advanced Installer.

1.  Localice el paquete final de distribución (`setup.exe` y `.msi`).
2.  Haga clic derecho sobre **`setup.exe`** y seleccione **"Ejecutar como administrador"** para evitar errores de permisos.
3.  Siga el asistente de instalación.
